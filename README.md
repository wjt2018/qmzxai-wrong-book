# qmzxai 错题本

一个用于 [qmzxai.com](http://www.qmzxai.com) 在线刷题站的油猴脚本，**自动捕获答错的题目**并支持一键导出复习。

零额外网络请求——脚本只监听浏览器自己的流量，服务端看到的访问模式与你手动刷题完全一致。

## 功能

- ✅ **自动记录错题**：题干、所有选项、你的答案、正确答案、解析、所在章节，全部捕获
- ✅ **按章节分组**：导出时按课程章节组织，方便针对薄弱点复习
- ✅ **三种导出格式**：
  - **Markdown** — 看着舒服，可贴到 Notion / Obsidian / 印象笔记
  - **JSON** — 程序化处理 / 二次开发用
  - **Anki TSV** — 直接导入 Anki，每道错题变成一张正反卡片，章节作为标签
- ✅ **同题反复错**：只保留最新一次记录
- ✅ **答对不自动移除**：是否清理由你决定

## 安装

### 桌面浏览器（Chrome / Edge / Firefox）

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展
2. 点 Tampermonkey 图标 → **管理面板** → 左上角 **+** 新建脚本
3. 把默认模板**全部删除**，将本仓库的 [`qmzxai_wrong_book.user.js`](./qmzxai_wrong_book.user.js) 内容**完整复制粘贴**进去
4. `Ctrl+S` 保存

**或者一键订阅**（油猴会自动检测更新）：

```
https://raw.githubusercontent.com/wjt2018/qmzxai-wrong-book/main/qmzxai_wrong_book.user.js
```

把这个 URL 粘到油猴"实用工具 → 从 URL 安装"即可。

### 安卓手机

Android 原版 Chrome 不支持扩展，请改用以下任一浏览器：

| 浏览器 | 怎么装 |
|---|---|
| **Firefox for Android** | 装 Firefox → 设置 → 附加组件 → 搜 Tampermonkey 装上 → 同桌面流程 |
| **Via 浏览器** | 装 Via → 菜单 → 设置 → 脚本管理 → "+" → 粘贴脚本内容（无需 Tampermonkey） |
| **X 浏览器** | 同 Via，自带脚本管理 |

### iOS

iOS 上推荐 **Userscripts** App（开源免费，Safari 扩展），装完后把脚本添加进去即可。

## 使用

1. 登录 qmzxai.com 进入刷题页面（必须**刷新一次**才能让脚本生效）
2. 页面右下角会出现红色 **📝 错题本 (0)** 浮动按钮
3. **正常做题**——错了就被自动记录，按钮上的计数会涨
4. 点按钮展开菜单，选择需要的导出格式：

```
⬇️ 下载 Markdown（按章节分组）
⬇️ 下载 JSON
⬇️ 下载 Anki 卡片 (.txt)
ℹ️ 统计信息          ← 查看各章节错题数
🗑️ 清空错题本
```

### Anki 导入

下载到的 `qmzxai_wrong_anki.txt` 直接拖进 Anki 桌面端的"文件 → 导入"。Anki 会自动识别字段（正面 / 背面 / 标签），章节名作为标签可在侧栏过滤复习。

### Markdown 导出示例

````markdown
# qmzxai 错题本
共 12 题，分布在 3 个章节

## 目录
1. 1.1基本要求部分职业道德之基本知识 — 4 题
2. 2.1基本要求部分基础知识之计算机系统基础知识 — 5 题
...

---

## 1.1基本要求部分职业道德之基本知识

### 1. `[单选]` 职业道德对职业者个人的作用是？
- A、保障职业者的合法权益 ❌(我选)
- B、促进职业者的全面发展 ✅
- C、决定职业者的职业地位
- D、提高职业者的社会声望

- **我的答案**：A、保障职业者的合法权益
- **正确答案**：B、促进职业者的全面发展
- *最后错误于 2026/6/10 下午3:21:05*
````

## 注意事项

- **必须刷新页面**脚本才会生效（XHR 钩子需要在站点 JS 之前注册）
- 装脚本**之前**做的题没有缓存，再做一次才会被完整记录
- 数据存在浏览器 `localStorage`，**换浏览器 / 清缓存会丢失**，建议定期导出 JSON 备份
- 不同设备的错题本独立保存，不会自动同步

## 工作原理

1. 在站点的 `main.js` 之前注入 XHR 钩子（`@run-at document-start`）
2. 拦截 `backend/admin/getRandomQuestion` 响应，缓存每道题的完整内容（含 `rightFlag` 正确答案标记）
3. 拦截 `backend/admin/submitResult` 请求 + 响应，识别答错事件
4. 将"题目缓存 + 你的选择 + 正确答案"合并写入 `localStorage`
5. 浮动按钮提供导出/清理 UI

整个过程**没有额外的网络请求**，仅是 client-side 数据加工。

## 隐私

- 所有数据**只存在你本地浏览器的 localStorage**
- 脚本不向任何第三方服务发送数据
- 源码全部公开（共 ~300 行 JS），欢迎审计
